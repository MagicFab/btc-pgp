In order to use BTC/PGP, you must generate a bitcoin address, fund that address, and broadcast a transaction. Broadcasting a transaction will reveal your public key, giving others the ability to send you messages encrypted to your public key. Because this is a proof of concept, this app uses testnet, and address generation, funding, and transaction broadcast can all be done in the browser below.

This app lets you send PGP encrypted mail with the bitcoin blockchain. Mail is encrypted using bitcoin public keys, and sent to recipients using OP_RETURN transactions. This is just an experiment, and uses the Testnet blockchain. To use this app, you need a private key (to send and decrypt messages), a small amount of bitcoin (to broadcast messages to recipients), and a special BTC-PGP transaction (to broadcast your pubilc key).
